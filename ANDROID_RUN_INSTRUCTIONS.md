# Инструкция: Как запустить приложение в Android Studio

Поскольку я пишу код, но не могу его запустить, тебе нужно будет сделать это самостоятельно. Вот пошаговая инструкция:

## 1. Подготовка
Убедись, что у тебя установлен **Android Studio** (версия Hedgehog или новее рекомендуется).

## 2. Открытие проекта
1. Запусти Android Studio.
2. Нажми **Open**.
3. Найди и выбери папку: `/home/m/Projects/make_story_ai/android` (именно папку `android`, а не корень проекта!).
4. Дождись окончания **Gradle Sync** (Android Studio будет скачивать зависимости, это может занять пару минут).

## 3. Настройка эмулятора
1. Открой **Device Manager** (иконка телефона справа вверху).
2. Если нет виртуального устройства, нажми **Create Device**.
3. Выбери любой Pixel (например, Pixel 7).
4. Выбери System Image (рекомендую **API 33** или **API 34**).
5. Нажми Finish.

## 4. Подключение к Backend
⚠️ **Важно:** Эмулятор Android работает в своей виртуальной сети.
*   `localhost` для эмулятора — это сам телефон.
*   Ваш компьютер (где запущен Python Backend) доступен по адресу: `10.0.2.2`.

Убедись, что в коде Android (я скоро это добавлю в Retrofit конфиг) Base URL установлен как `http://10.0.2.2:8000/`.

## 5. Запуск
1. Убедись, что Python backend запущен в терминале:
   ```bash
   cd ~/Projects/make_story_ai
   source venv/bin/activate
   uvicorn backend.app.main:app --host 0.0.0.0 --port 8000 --reload
   ```
2. В Android Studio нажми зеленую кнопку **Run (▶)**.
3. Приложение должно открыться на эмуляторе.

## Возможные проблемы
*   **Ошибка соединения (Network Error):** Проверь, что backend запущен и доступен.
*   **Cleartext Traffic not permitted:** Я добавлю разрешение в AndroidManifest, чтобы можно было ходить по http (не https) при разработке.
